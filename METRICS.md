# Metrics

Draft status: started on Day 1.

The North Star metric is qualified audits completed, where "qualified" means an audit with enough tool and spend data to produce a credible recommendation. This fits a B2B lead-generation tool better than daily active users because most teams will use it only when reviewing spend or after pricing changes.

Three input metrics drive it: started audits, completed audits, and captured reports. A fourth metric, high-savings audits above $500/month, matters for Credex because those users are the strongest consultation candidates.

The first instrumentation should track form start, tool row added, audit completed, email captured, public link copied, and consultation CTA clicked. The pivot trigger is simple: if 100 completed audits produce fewer than 5 high-savings leads or fewer than 2 consultation requests, either the target audience is wrong or the audit is not surfacing painful enough problems.
